## Цветовая схема

- **Неоновый синий (Cyan)** - заголовки, акценты, важная информация
- **Неоновый фиолетовый (Magenta)** - рамки, границы, вторичные элементы
- **Зеленый** - доходы, успешные операции, прогресс
- **Красный** - расходы, ошибки, предупреждения
- **Желтый** - предупреждения
- **Белый** - основной текст
- **Серый** - второстепенная информация

## Особенности дизайна

1. **Неоновые рамки** - все элементы в стильных рамках
2. **Градиентные эффекты** - переходы цветов для визуальной глубины
3. **ASCII-арт** - красивые символы для оформления
4. **Прогресс-бары** - визуальное отображение прогресса
5. **Уведомления** - выделенные блоки для важной информации
6. **Структурированная информация** - четкое разделение секций
